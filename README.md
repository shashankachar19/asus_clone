# AI Frontend Challenge: The Zero-Shot Benchmarks

This repository is a head-to-head comparison of the frontend architectural capabilities of three leading AI models: **Gemini**, **Claude**, and **v0**.

The goal is to evaluate how accurately these models can replicate a complex, high-end hardware website (ASUS TUF/ROG) using a **Zero-Shot Prompting** strategy. 

> **Zero-Shot Definition:** Only one single, detailed prompt was provided to each model. No follow-up corrections, no iterative styling, and no manual bug fixes were allowed.

To be honest, the code generated by all three models is remarkably high-quality for a single-turn interaction. Each model prioritized different aspects:

* **Claude**: Traditionally strong at logical component structure and clean Tailwind utility usage.
* **v0**: Highly specialized in React-centric components and modern UI patterns.
